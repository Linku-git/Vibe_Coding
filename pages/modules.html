The Master Workflow

This is the actual process the course teaches. Follow it every time.

01

Start with Idea

Write a plain-language project idea.

02

Generate PRD

Use ChatGPT to create prd.md. Refine until implementation-ready.

03

Create Clean Repo

Add Docs/, runtime setup, .env.example, CLAUDE.md, tests.

04

Generate Agent Rules

Create agents.md and/or CLAUDE.md for disciplined AI behavior.

05

Generate Supporting Docs

Architecture, progress, routes, pages, testing strategy.

06

Generate Session Files

Break project into atomic sessions (6-10 simple, 15-30 complex).

07

Open in AI Tool

Open repo in Claude Code, Codex, or Antigravity. Load docs first.

08

Run One Session Only

Never ask for the whole project in one prompt.

09

Test Locally

Run app, run tests, inspect errors.

10

Update Docs

Every session must leave the repo better documented.

11

Continue to Next Session

Only after current session is stable. Repeat from step 08.
