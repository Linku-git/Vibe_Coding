Setup Guides

Installation and configuration guides for every tool in the Vibe Coding workflow.

Installation

Install via npm (requires Node.js 18+)
npm install -g @anthropic-ai/claude-code

First Run

1
Open your terminal in your project folder
cd my-project
2
Run Claude Code
claude

On first run, it will ask you to authenticate with your Anthropic API key.

3
Create CLAUDE.md

Create a CLAUDE.md file at your project root. This is your project-level instruction file that Claude reads automatically.

4
Create custom commands (optional)
mkdir -p .claude/commands

Add markdown files like .claude/commands/run-session.md to create slash commands.

API Key Setup

You need an Anthropic API key. Get one at console.anthropic.com:

  1. Create an account at console.anthropic.com
  2. Go to API Keys section
  3. Click "Create Key" and copy it
  4. When Claude Code prompts for authentication, paste your key

VS Code Integration

Install the "Claude Code" extension from the VS Code marketplace. It adds a panel where you can interact with Claude directly inside your editor.

Verify Installation

Test that everything works
claude --version
claude "What files are in this directory?"

Basic docker-compose.yml for Full-Stack Projects

docker-compose.yml
version: "3.8"
services:
  db:
    image: postgres:16
    environment:
      POSTGRES_DB: myapp
      POSTGRES_USER: user
      POSTGRES_PASSWORD: password
    ports:
      - "5432:5432"
    volumes:
      - pgdata:/var/lib/postgresql/data

  redis:
    image: redis:7-alpine
    ports:
      - "6379:6379"

volumes:
  pgdata:

Run with: docker compose up -d — Stop with: docker compose down

SSH Key Setup for GitHub

1
Generate SSH key
ssh-keygen -t ed25519 -C "you@example.com"

Press Enter for default file location. Optionally set a passphrase.

2
Copy the public key
cat ~/.ssh/id_ed25519.pub

Copy the output.

3
Add to GitHub

Go to GitHub → Settings → SSH and GPG keys → New SSH key → Paste and save.

4
Test the connection
ssh -T git@github.com

How MCP Servers Work

MCP servers are configured in .claude/settings.json. Each entry runs a small server that exposes tools to your AI agent. You need Node.js installed for npx commands.

GitHub MCP

.claude/settings.json
"github": {
  "command": "npx",
  "args": ["-y", "@modelcontextprotocol/server-github"],
  "env": {
    "GITHUB_PERSONAL_ACCESS_TOKEN": "ghp_your_token"
  }
}

Get token: GitHub → Settings → Developer Settings → Personal Access Tokens → Generate New Token. Select scopes: repo, read:org.

PostgreSQL MCP

.claude/settings.json
"postgres": {
  "command": "npx",
  "args": ["-y", "@modelcontextprotocol/server-postgres"],
  "env": {
    "DATABASE_URL": "postgresql://user:pass@localhost:5432/mydb"
  }
}

Requires: PostgreSQL running locally (see Docker guide above for easy setup).

Playwright MCP (Browser Automation)

.claude/settings.json
"playwright": {
  "command": "npx",
  "args": ["-y", "@anthropic-ai/mcp-playwright"]
}

No API key needed. Automatically launches a browser for UI testing.

Slack MCP

.claude/settings.json
"slack": {
  "command": "npx",
  "args": ["-y", "@anthropic-ai/mcp-slack"],
  "env": {
    "SLACK_BOT_TOKEN": "xoxb-your-token"
  }
}

Get token: Go to api.slack.com/apps → Create New App → OAuth & Permissions → Add scopes → Install to Workspace → Copy Bot Token.

Full Combined Example

.claude/settings.json — complete
{
  "mcpServers": {
    "github": {
      "command": "npx",

    },
    "playwright": {

    }
  }
}

Python (for Flask/FastAPI/Django backend)

1
Install Python 3.12+

Download from python.org. On Mac: brew install python. On Linux: sudo apt install python3.

2
Create a virtual environment
python -m venv .venv && source .venv/bin/activate

Windows: .venv\Scripts\activate

3
Install dependencies
pip install -r requirements.txt
