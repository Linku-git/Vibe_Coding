Claude Code Deep Dive

Claude Code is unique because it lives in your terminal. It reads your files, runs your commands, and follows project rules defined in CLAUDE.md. Key differentiators:

  • CLAUDE.md files — like .cursorrules but hierarchical (project, folder, user level)
  • Slash commands — custom /commands for repeatable tasks (e.g., /session-run)
  • Hooks — automated shell commands on tool events (pre-edit linting, post-commit checks)
  • Memory system — remembers user preferences, project decisions across sessions
  • Subagents — spawns focused agents for parallel research or implementation
  • Worktrees — git worktree isolation for safe parallel branches

Recommendation for This Course

  • Use ChatGPT for planning and documentation generation
  • Use Claude Code as the primary structured coding tool
  • Use Codex for parallel agent workflows
  • Optionally use Antigravity for autonomous or experimental flows
