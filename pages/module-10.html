Execution Prompt — Codex / Antigravity

Read:
- Docs/prd.md
- Docs/agents.md
- Docs/architecture.md
- Docs/testing-strategy.md
- Docs/sessions/session-03.md

Complete only Session 03.

Requirements:
- follow the PRD and agent rules
- make small safe changes
- do not implement work from later sessions
- run relevant local tests
- report files changed
- report errors and fixes
- update:
  - Docs/progress.md
  - Docs/api-routes.md if changed
  - Docs/frontend-pages.md if changed
  - Docs/readme.md if setup changed
  - Docs/sessions/session-03.md with notes and results

Execution — Claude Code

With Claude Code, your CLAUDE.md already contains the rules. You can be more concise:

Read Docs/sessions/session-03.md and complete it.

Follow all rules in CLAUDE.md.
Only implement session 03 tasks.
Run tests and update docs when done.

Or create a custom slash command:

.claude/commands/run-session.md
Read Docs/sessions/session-$ARGUMENTS.md

Complete only this session following CLAUDE.md rules.
Run all relevant tests.
Update Docs/progress.md and the session file with:
- files changed
- tests run and results
- errors found and fixes applied
- pending issues

Then run: claude "/project:run-session 03"

Critical Rule: Always constrain the AI to one session, one goal area, one validation cycle. That is how you avoid chaos.
